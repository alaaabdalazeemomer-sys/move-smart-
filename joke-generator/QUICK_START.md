# 😂 Joke Generator - Quick Start

## One-Command Setup

```bash
cd joke-generator
npm install
npm run dev
```

Then open: **http://localhost:5173**

---

## Features

✨ **Random Jokes** - Fresh jokes every click  
✨ **Categories** - Pick your joke type  
✨ **Copy & Share** - Share jokes on social media  
✨ **Beautiful UI** - Modern, responsive design  
✨ **Real API** - Uses JokeAPI (https://jokeapi.dev)  

---

## Available Commands

```bash
# Development
npm run dev

# Build for production
npm run build

# Preview production
npm run preview
```

---

## How It Works

1. Click **"Get Random Joke"** button
2. Wait for joke to load
3. Read and laugh 😂
4. Click **"Copy"** to copy joke
5. Click **"Share"** to share on Twitter, Facebook, or Email
6. Choose different categories to try different joke types

---

## Joke Categories

- 🎯 **Any** - Random joke from all categories
- 😄 **General** - General jokes
- 💻 **Programming** - Tech and programming jokes
- 🔔 **Knock-Knock** - Classic knock-knock jokes
- 🌑 **Dark** - Dark humor
- 👻 **Spooky** - Spooky jokes
